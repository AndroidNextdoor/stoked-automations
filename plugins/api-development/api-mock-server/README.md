# API Mock Server Plugin

Create mock API servers from OpenAPI specs with realistic data and customizable responses.

## Installation

```bash
/plugin install api-mock-server@stoked-automations
```

## Usage

```bash
/create-mock-server
```

## Features

- OpenAPI-based generation
- Realistic fake data
- Stateful mocking
- Latency simulation
- Error scenarios
